# JEE Commander V4 🚀

A high-performance, dark-themed preparation dashboard engineered to track execution metrics, analyze productivity velocity, and maintain rigid daily discipline for the Joint Entrance Examination (JEE).

## ✨ Features
- **Intelligence System Metrics:** Automated analytics engine that tracks historical inputs to dynamically flag your strongest subjects and critical weaknesses.
- **Productivity Efficiency Grading:** Calculates live question-solving speed (Questions per Hour) and maps performance to active operational ranks (`🥇 S-Tier`, `⚡ A-Tier`, `📈 B-Tier`).
- **Mastery Routine Log:** Built-in tactical habits designed specifically for competitive excellence (e.g., *5 AM Club*, *Error Logs*, *Digital Minimalism*).
- **Responsive Chart Visualizations:** Fully dynamic tracking using Chart.js to map temporal study distribution, numerical practice metrics, routine mastery streaks, and mock test scores relative to an auto-scaling target curve.
- **Privacy First Architecture:** Local-first operation utilizing your browser's native `localStorage`. Your academic metrics are completely private and stay exclusively on your device.

## 🛠️ Tech Stack
- **Interface Structure:** HTML5 / Semantic Layout
- **Theming & Design:** Custom CSS3 Variables (Optimized Dark Mode Ecosystem)
- **Data Engine:** Vanilla Javascript (ES6+)
- **Analytics Visualization:** [Chart.js](https://www.chartjs.org/) via CDN

## 🚀 Deployment & Installation
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. *Alternative:* Host it for free via **GitHub Pages** to track your metrics across mobile devices.
